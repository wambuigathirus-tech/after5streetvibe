# After 5 Street Vibe Website

A static website for the After 5 Street Vibe creative movement in Nakuru.

## Files

- `index.html` — main website page
- `styles.css` — styling for layout and visual design
- `script.js` — dynamic event cards and gallery
- `assets/a5.png` — logo
- `assets/tuko kadi.webp` — gallery image
- `package.json` — Node dependencies
- `server.js` — local server

## Setup

1. Copy your logo to `assets/a5.png`.
2. Add gallery images to `assets/` and update `script.js`.
3. Open `index.html` in a browser or run `python -m http.server 8000` and visit `http://localhost:8000`.

## Notes

- The website features animated elements like floating bubbles and glowing geometric lines for visual appeal.
- Add event photos or past activity images into the `assets/` folder and update the HTML if you want image galleries.
